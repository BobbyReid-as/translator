# Evaluation Report: Multi-Language Translation Pipeline

## Overview
This project implements an automated translation pipeline using HuggingFace’s **MarianMT** models. The system features an auto-detection mechanism for source languages and a custom **Gradio** web interface for user interaction. To assess the model's accuracy, a comprehensive evaluation was conducted across five language pairs, comparing results against a **Google Translate** baseline using the **SacreBLEU** metric.

## Methodology
* **Models**: Utilized specialized `Helsinki-NLP/opus-mt` models for French, German, Spanish, and Italian. Portuguese was handled by the `opus-mt-ROMANCE-en` model.
* **Dataset**: Evaluated 20 diverse sentences per language pair (100 total) covering everyday topics like weather, travel, and technology.
* **Metrics**: Used **SacreBLEU** to calculate corpus-level and sentence-level scores. BLEU scores range from 0 to 100, where higher scores indicate closer alignment with human-translated reference sentences.

## Results Summary
The following table summarizes the corpus-level BLEU scores generated by the `evaluate.py` script.

| Language Pair | MarianMT BLEU | Google Translate BLEU | Difference |
| :--- | :--- | :--- | :--- |
| **Spanish → English** | **80.69** | 90.50 | -9.81 |
| **German → English** | 79.47 | 81.94 | -2.47 |
| **Portuguese → English** | 70.71 | 71.99 | **-1.28** |
| **French → English** | 58.00 | 79.49 | -21.49 |
| **Italian → English** | 56.00 | 66.52 | -10.52 |

## Performance Analysis
* **Highest Performance**: The **Spanish (ES→EN)** pair achieved the highest BLEU score for MarianMT at **80.69**. Multiple sentences, such as "Me gusta mucho leer libros de historia," achieved perfect **100.0** scores.
* **Most Competitive**: The **Portuguese (PT→EN)** model showed the closest performance to the commercial baseline, trailing Google Translate by only **1.28 points**.
* **Accuracy Gaps**: The **French (FR→EN)** pair showed the largest gap (-21.49). While simple sentences like "Cette ville est connue pour ses musées" scored perfectly, the model struggled with nuanced phrasing where Google Translate provided more natural outputs.
* **Observation**: Models generally performed better on short, declarative sentences but showed variance when handling contractions or slight grammatical shifts.

## How to Reproduce
This project utilizes **`uv`** for extremely fast and reproducible environment management.

1.  **Sync Environment**:
    ```bash
    uv sync
    ```
2.  **Run Translation App**:
    ```bash
    uv run python main.py
    ```
3.  **Run Evaluation**:
    ```bash
    uv run python evaluate.py --with-google
    ```
